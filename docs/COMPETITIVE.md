# 競品分析與定位（2026-07-11，三路 agents 實查 viAct / Intenseye / Verkada / Samsara / Procore）

## 一句話結論

**我們是「說故事最好的產品導覽」，他們是「證據最厚的商業提案」。**
敘事結構、產品展示的誠實度（真畫面＋活元件）、在地語感我們全面領先；
社會證據（客戶 logo、成果數字、案例）是唯一結構性落後，需要真實實績累積後補上。

## 我們贏的（已內建）

| 面向 | 我們 | 競品 |
|------|------|------|
| 敘事 | 三幕故事線（看得見/攔得住/說得清） | 全部是功能清單式排版 |
| 產品展示 | 真實系統畫面＋六個活元件＋互動告警 demo | 靜態截圖或宣傳 mockup |
| CTA 摩擦 | 15 分鐘、不綁約、用現有攝影機（= Intenseye 最佳實踐） | Verkada/Samsara CTA 分散 |
| 在地化 | 職安署語彙、勞檢鉤子、臺灣職安卡、台灣機房、「安全是回家唯一的路」 | 全球玩家碰不到 |
| 誠實差異化 | 「本頁畫面皆擷取自真實系統」「工地睡了，系統醒著」 | 無人用夜間真畫面當賣點 |

## 我們輸的（backlog，需真實數據）

1. **社會證據為零** vs Verkada「30,000+ 組織」、Intenseye 27 logo、viact 具名客戶——
   → 有第一批客戶後：實績帶（N 個工地、監測時數、攔阻次數）＋去識別化實測卡。數字必須真實可答辯。
2. **成果數字**（-95% 事故、-81% 風險）→ 導入 30 天後開始收集：告警下降 %、巡場工時節省。
3. **Persona 分眾帶**（工安主管/工務主管/業主各一句）→ 版面已預留，之後加。
4. **次要轉換出口**（導入檢核表 PDF 換 email）→ 資產做好後加。
